Getting Started

  1. Use the provided template or create your own file(s). Supported formats include: .xlsx, .xls, .ods, .csv
  2. Prepare your spreadsheet(s) with questions. Make sure each row contains all the required fields based on the question type (see below).
  3. Upload your file(s) by selecting them or dragging them into the designated area.
  4. Click Download to get your Moodle®-compatible XML file.

Formatting Your Spreadsheet

Use the provided template or create your own with each row representing a question (Column Name = Cell Value):

  • Multiple Choice Row:
    Type = "MC", Title, Question, OptionA-D, Correct (e.g., "A")
  • True/False Row:
    Type = "truefalse", Title, Question, Correct ("TRUE" or "FALSE")
  • Short Answer Row
    Type = "shortanswer", Title, Question, Correct1â€“4 (answer text/alternatives), UseCase (OPTIONAL, "1" for case-sensitive)

You can store questions, of any type, in several sheets, organized any way you like.
