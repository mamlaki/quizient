About Quizient

What is Quizient?

Quizient is a simple tool designed to help educators and administrators convert spreadsheet-based question banks into the Moodle® XML format. Whether you're an educator managing a single course or an administrator managing an entire site, Quizient helps save time and is very easy to use.

Why Does This Exist?

During my Bachelor of Technology (BTech) degree capstone project, my team recommended Moodle® as part of a custom solution for our project sponsors, who were looking for a reliable platform for distirbuting and managing emergency services aptitude tests. Moodle was a strong fit, especially for its question bank capabilities.

After completing the degree, I built Quizient to support the sponsors—and others like them—by making it easy to convert questions organized in spreadsheets into Moodle's XML format, allowing them to be imported seamlessly.

Who's This For?

  • Instructors using Moodle for quizzes or exams
  • Course coordinators managing large question banks
  • Administrators or IT staff managing Moodle instance(s) for an organization
  • Anyone who wants a faster way to bulk import questions into Moodle

How It Works

  1. Upload your spreadsheet (.xlsx, .xls, .ods, or .csv)
  2. Each row becomes a question (Multiple Choice, True/False, Short Answer)
  3. Quizient generates a Moodle-compatible XML file
  4. Download the XML file and import it into Moodle

Feel free to visit the GitHub repo to take a look at the code itself or to run the app locally!

Features

  • Supports multiple question types
  • Accepts alternate short answer responses
  • Wildcard support for partial short answer matches
  • Convert several spreadsheets at once
  • Built-in validation for common errors
